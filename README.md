LESSCSS mixins file of vendor prefixes for CSS3

###Use:

	#someElement {
		.box-shadow();
		// or
		.transition(all, 1s);
	}

###Mixins available

* .border-radius();
* .outline-radius();
* .box-shadow();
* .transition();
* .transform();
* .box();
* .flex();
* .resize();
* .linear-gradient();
* .double-borders();
* .triple-borders();
* .colomns();
* .box-sizing();
* .text-shadow();
